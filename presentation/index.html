01 / DRAFTING

While the model is still composing its reply.

A chat surface where the user has just hit send. The model's reasoning steps stream in below; dots for the headline status, arc and breathe for the sub-steps. Calm pacing, unintrusive — the spinner means "still here, still working."

dots arc breathe
02 / TOOL-USE STREAM

Different glyph per tool, so the user can read the activity.

When an agent is searching, querying a DB, running tests, and formatting code in parallel, a single global spinner says nothing. Pick a distinctive glyph per tool, and the timeline becomes legible at a glance.

sparkle helix fillsweep scan
03 / BACKGROUND RUN

Terminal-native, even when there's no terminal.

For long-running agent tasks — applying migrations, streaming tests, waiting on CI — a CLI feel grounds the UI in the developer's mental model. Same braille glyphs they already trust in ora, spin, cli-spinners.
